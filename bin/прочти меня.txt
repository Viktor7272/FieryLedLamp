Текущая версия / Current version : FieryLamp_v2.15_web_v2.0  ( gunner47_v2.15_web_v2.0_88in1 )

Бинарные файлы для загрузки по воздуху через web интерфейс.
Все настройки по умолчанию в соответствии со схемой и файлом constant.h . 

ВНИМАНИЕ!  Бинарники сделаны под конфигурацию Flash Size "4MB (FA:2MB ; OTA~1019KB)"

Чтобы было проще:
  -  Пропишите в файле data/config.json имя и пароль вашей сети и espMode поменяйте с 0 на 1.
  - Прошивайте сначала Firmware , а затем FileSystem.
